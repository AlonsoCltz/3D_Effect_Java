Using 2D tools to draw 3D effect simulating what Doom does

- Storing the Map
Store the map as text file inside /mapStorage
Use number to present color (0 is reserved for floor)

- Mini Map
The mini map will show the explored available paths
Click on the mini map to view it larger, click again to close it

- Quick test
Run Game.java, it will retreive the default graph (default maze)
Modify the graph retreive by changing the graph ID
(or simpler, change the file name in GraphStorage.java)
